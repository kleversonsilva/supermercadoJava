# Projeto - Api de Um software para supermercado

## Api simples, cuja a funcionalidade é cadastrar produtos e realizar vendas dos produtos:

### Especificações principais

* Java
* GIT
* Github

- ***Clientes***
-  listar
-  comprar
-  ver carrinho
-  finalizar compra


- ***Supermercado***
  - cadastrar produto
  - remover produto
  - finalizar compra
